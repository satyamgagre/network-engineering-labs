## Topics Covered

- Switching (VLANs, Trunking, VTP, EtherChannel)
- Layer 2 Security (Port Security, DHCP Snooping, DAI, IPSG)

- RIP, EIGRP, OSPF, Multi-Area OSPF, BGP
- DHCP, DNS, FTP, Email and Web Services
- ACLs
- NAT / PAT
- HSRP
- IPv6
- Cisco ASA Firewall
- IPSec VPN
