A complete Cisco Packet Tracer lab portfolio covering switching, routing, IPv4 and IPv6, network services, WAN, wireless, VoIP, security, Cisco ASA firewall, and VPN concepts.

## About This Repository

This repository contains 75 hands-on networking labs built in Cisco Packet Tracer. The labs are organized as a practical learning path for CCNA-level networking skills, from basic device configuration to advanced firewall, VPN, wireless, and network service scenarios.

Each lab includes the main Packet Tracer activity file, and most labs also include a topology screenshot and lab notes or documentation.

## Skills Covered

- Basic switch and router configuration
- VLANs, trunks, native VLANs, and VTP
- Telnet and SSH remote access
- EtherChannel using PAgP, LACP, and manual mode
- Port security, DHCP snooping, DAI, IPSG, BPDU Guard, and Root Guard
- Inter-VLAN routing using router-on-a-stick and multilayer switches
- DHCP, DNS, web, email, and FTP services
- Static routing, floating static routes, RIP, EIGRP, OSPF, Multi-Area OSPF, and BGP
- Standard ACLs, extended ACLs, and VTY access control
- NAT, PAT, HSRP, IPv6, DHCPv6, RIPng, EIGRPv6, and OSPFv3
- PPP, GRE, IPSec VPN, WLAN, WLC, VoIP, and Cisco ASA firewall configuration
